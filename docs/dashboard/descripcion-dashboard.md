# Dashboard de Monitoreo de Proyectos en Power BI

Este dashboard fue desarrollado para visualizar el estado de los proyectos a partir de un archivo Excel actualizado automáticamente mediante Power Automate.

## Indicadores principales

- Total de proyectos activos
- Total de proyectos finalizados
- Nuevos proyectos
- Proyectos por dirección
- Porcentaje de avance
- Estado general de los proyectos

## Fuente de datos

La fuente de datos utilizada es un archivo Excel almacenado en OneDrive, actualizado automáticamente con información proveniente de Fiix.

## Objetivo del dashboard

Facilitar el monitoreo de proyectos y apoyar la toma de decisiones mediante información actualizada y visualizaciones interactivas.

## Nota

Por motivos de confidencialidad, no se incluye el archivo original ni enlaces internos al dashboard.
