# Diccionario de Datos

Este documento describe los campos utilizados en el archivo de proyectos de ejemplo.

| Campo | Descripción | Tipo de dato |
|---|---|---|
| id_proyecto | Identificador único del proyecto | Texto |
| nombre_proyecto | Nombre del proyecto | Texto |
| direccion | Dirección o ubicación relacionada | Texto |
| estado | Estado actual del proyecto | Texto |
| fecha_inicio | Fecha de inicio del proyecto | Fecha |
| fecha_fin | Fecha de finalización del proyecto | Fecha |
| porcentaje_avance | Porcentaje de avance del proyecto | Número |
| responsable | Responsable asignado al proyecto | Texto |
| ultima_actualizacion | Fecha de última actualización | Fecha |

## Nota

Los campos utilizados son genéricos y no contienen información confidencial.
