# Consideraciones de Seguridad y Confidencialidad

Este repositorio no incluye información sensible ni datos reales de la organización.

## Información no incluida

No se incluyen:

- Correos reales
- Credenciales
- Contraseñas
- Tokens
- Enlaces privados de OneDrive
- Archivos internos reales
- Nombres reales de proyectos confidenciales
- Información privada de responsables
- Datos empresariales sensibles
- Configuraciones internas de Power Automate

## Datos utilizados

Los datos incluidos en este repositorio son ficticios o anonimizados y tienen como único propósito demostrar la lógica general de la solución.

## Buenas prácticas aplicadas

- Uso de datos de ejemplo.
- Documentación del flujo automatizado.
- Separación entre datos, documentación e imágenes.
- Protección de información sensible.
